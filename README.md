
# 5 Minutes Games 👾
Este proyecto es una plataforma web de videojuegos cortos. Desarrollado por los alumnos de 5to año en Computación, E.T. N°12.

Construida con HTML5, CSS3 y JavaScript, importando los juegos desde Unity.
